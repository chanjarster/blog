**Sort Big File**: Imagine you have a 20 GB file with one string per line. Explain how you would sort the file.

Hints: #207

## 解法

拆成若干小文件，比如每个文件100M，对其做快排，然后把这些小文件合并起来（合并两个有序数组）。
